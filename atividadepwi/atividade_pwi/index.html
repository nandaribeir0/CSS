O QUE É CSS?

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas.

É a estilização de uma página, e permite que você estilize tudo em um arquivo diferente, criando assim o estilo separadamente e depois faça integração do arquivo CSS na parte superior da marcação HTML.

É uma boa solução para aprimorar a estética dos sites. Por meio dele, é possível separar formatos de conteúdos e tambem definir como serão exibidos em cada página.

O CSS é um código em que você pode fazer alterações rápidas de layout, como alterar cores e fontes. Esse código proporciona facilidade de personalização e também ajuda a diminuir a repetição de conteúdo na estrutura do código.

Ele permite que você estilize tudo em um arquivo diferente, criando assim o estilo separadamente. E só depois faça integração do arquivo CSS na parte superior da marcação HTML. Isso mantém a marcação HTML limpa e fácil de manter.

ESPECIFICIDADE

Especificidade CSS é um tipo de “poder de escolha”, que ajuda os navegadores a decidir qual valor da propriedade CSS terá precedência e será aplicado aos elementos.

Seletores Primeiramente, vamos entender quais são os seletores mais específicos. Na lista abaixo, os seletores estão ordenados do mais específico ao menos específico.

  • Ids #id { ... }
  • Classes, atributos e pseudo-classes .classe { ... }
  • Elementos e pseudo-elementos div { ... }

Para calcular a especificidade do CSS, primeiro escolhemos o seletor e depois calculamos quantas Ids e classes (incluindo pseudo-classes e atributo) e vários elementos existem. Depois de conhecê-los, escrevemos na ordem: abc.

Nós os comparamos movendo da esquerda para a direita. Assim que encontramos um ponto (a / b / c) maior que o outro, paramos por aí.

Assim sabemos quem se tornou o vencedor com mais poderes, ou seja, com maior ordem de precedência.

SOBRE PSEUDO-CLASSES

Uma pseudo-classe CSS é uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado. Por exemplo, :hover pode ser usado para alterar a cor de um botão quando o usuário passar o cursor sobre ele.

Pseudo-classes Dinâmicas

As pseudo-classes dinâmicas controlam os estados dos elementos. Abaixo, vão alguns deles:

  • hover: quando passamos o mouse em cima do elemento.
  • active: quando ativamos o elemento. Por exemplo, quando clicamos em um link e não soltamos o botão do mouse.
  • visited: quando o link é visitado.
  • focus: quando um elemento recebe foco. Muito utilizado em campos de texto.

Pseudo-classes Estruturais

As pseudo-classes estruturais servem para selecionarmos um elemento da estrutura do código. Existem várias, por exemplo:

  • :first-child: seleciona o primeiro filho de um outro elemento.
  • :last-child: seleciona o último filho de um elemento.
  • :root: representa um elemento que é a raiz do documento. No HTML 4, é sempre a tag HTML.
  • :nth-child(): permite que selecionemos qualquer elemento no meio de um grupo de elementos.

SOBRE PSEUDO-ELEMENTOS

Um pseudoelemento CSS é uma palavra-chave adicionada a um seletor que permite que você estilize uma parte específica do elemento selecionado. Por exemplo, o pseudoelemento ::first-line aplica o estilo apenas na primeira linha de um parágrafo.

Sintaxe

Seletor::pseudo-elemento { propriedade: valor; }
(Você pode usar apenas um pseudoelemento em um seletor. Ele deverá aparecer depois da declaração de um elemento simples.)
